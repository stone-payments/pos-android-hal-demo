abstract class Pinpad(    val properties: Map<String, Any>,     val runtimeProperties: MutableMap<String, Any>,     val callbacks: PinpadCallbacks)
Content copied to clipboard

This class must be extended by the manufacturer. It's represents commands based on Biblioteca Compartilhada para Pinpad 108a.



There are blocking and non-blocking commands, non-blocking commands have a synchronous return, for blocking commands we have synchronous and asynchronous return.



For example blocking command: Using Pinpad.getCard method. This return synchronously PP_StartGetCard and return asynchronously PP_GetCard

For example non-blocking command: Using Pinpad.getInfo method. This return synchronously PP_GetInfo.
