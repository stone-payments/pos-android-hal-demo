Mifare

Class for handling Mifare cards, from search to activation and data operations. Note that data operations take "sector" as parameter. This means that block numbers go from 0 to 3, while sector numbers may vary between card types. In some platforms, block numbers are continuous, this means that the following formula should be applied: absBlock = (sector * 4) + block
