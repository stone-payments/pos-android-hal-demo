Parameters

serialNumber

Device Serial Number

isPosAndroid

The device is a Point Of Sale System

manufacturerName

Device Manufacturer Name

fullDeviceName

Full Device Name (deprecated, instead set 'modelName' and retrieve with 'manufacturerModel)

osBuildNumber

OS build number. Requirements:Sequential number; Only numbers and dots; For each field between the points the amount of digits must be constant; Max of four fields between the points (e.g. 1.0.0.0). Examples of accepted Kernel Version: "AAMMDD", "AA.MM.DD"

firmwareVersion

Device Firmware Version (deprecated, use osBuildNumber instead).

modelName

Device Model Name.

soSupportedExtension

so extension(with dot).

kernelVersion

Device Kernel Version. Requirements:Sequential number; Only numbers and dots; For each field between the points the amount of digits must be constant; Max of four fields between the points (e.g. 1.0.0.0). Examples of accepted Kernel Version: "AAMMDD", "AA.MM.DD"

cpuInfo

Returns the hardware information of the CPU.

iccReaderPosition

Position of ICC (Contact) Reader.

magReaderPosition

Position of MAG STRIPE Reader.

nfcReaderPosition

Position of NFC (Contactless) Reader.

keyboardType

Type of keyboard referenced in KeyboardType.

customResourceVersion

Custom Resource Version. Return CUSTOM_RESOURCE_UNKNOWNif not applicable.
